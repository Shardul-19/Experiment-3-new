# Experiment-3-new
Aim
To study the concept of tuples in Python and perform various operations such as creation of tuples, immutability checking, repetition, concatenation, sorting, unpacking, and comparison with lists.

Theory 

* A tuple in Python is a collection of elements that is ordered and immutable. Tuples are written using round brackets () and can store different data types such as integers, strings, and boolean values. Once a tuple is created, its elements cannot be changed, added, or removed.
* Tuples support various operations such as:
* Accessing elements using indexing
* Finding maximum, minimum, sum, and length
* Counting occurrences of elements
* Tuple unpacking
* Concatenation, which creates a new tuple (proving immutability)
* A list, on the other hand, is a collection that is ordered and mutable. Lists are written using square brackets [] and allow modification of elements after creation.
* Difference Between Tuple and List
* Tuple	List
* Uses round brackets ()	Uses square brackets []
* Immutable (cannot be changed)	Mutable (can be changed)
*Faster execution	Slower than tuple
* Uses less memory	Uses more memory
* No methods like append() or sort()	Has methods like append(), remove(), sort()
* Elements cannot be modified	Elements can be modified
* Safer for fixed data	Suitable for dynamic data

Algorithm

* Start the program.
* Create a tuple containing mixed data types and display it.
* Attempt to modify an element of a tuple to show that tuples are immutable.
* Create a tuple using repetition operator * and print it.
* Use id() function before and after tuple concatenation to show that a new tuple is created.
*  Create a tuple of marks and find maximum, minimum, total, length, and average.
* Sort the tuple using the sorted() function in ascending and descending order.
* Perform tuple unpacking and display the unpacked values.
* Use count() and membership operator in to monitor attendance.
* Stop the program.

Overall Conclusion

Thus, the experiment successfully demonstrates the use of tuples in Python. It proves that tuples are immutable, meaning their elements cannot be modified after creation. Various tuple operations such as repetition, concatenation, sorting using sorted(), unpacking, and counting elements were performed. The difference between tuples and lists was also studied, highlighting that tuples are more secure and memory-efficient for fixed data, while lists are suitable for dynamic data. Hence, tuples are best used when data integrity is important.
